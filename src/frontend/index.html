Technology Stack

Built with Enterprise DevOps

🐳 Docker

Containerization of all microservices with semantic versioning and Docker Hub registry.

⎈ Kubernetes

Orchestration with Deployments, Services, ConfigMaps, Secrets, and rolling updates.

🔧 Jenkins

Declarative multi-stage CI/CD pipeline: Checkout → Build → Test → Deploy → Notify.

⬡ GitHub Actions

Automated workflows for linting, Docker builds, and environment-specific deployments.

🟢 Node.js / Express

Backend runtime for all four microservices with REST API design.

📊 Prometheus + Grafana

Centralized monitoring, metrics collection, and alerting dashboards.
